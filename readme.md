# Slack bot episode #49

Let's build a Slack bot using BotKit framework! [here](https://youtu.be/Km-VjeB7z3w).

# Installation
```
git clone
npm i -g ngrok
npm install
node bot.js
ngrok http 3000
Create a new app on https://api.slack.com/apps
```
